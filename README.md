# duka-lanches

![](./img/logo.png)

Projeto desenvolvido exclusivamente para a hamburgueria Duka Lanches, localizada na Zona Leste de São Paulo.
O site tem como objetivo divulgar informações importantes como o seu início, localização, redes sociais e o cardápio, para obter um maior alcance de clientes e facilitar na hora dos pedidios.
