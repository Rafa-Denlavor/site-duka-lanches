# site-duka-lanches
Site criado para a hamburgueria Duka Lanches com o objetivo de obter um maior alcance de clientes e mostrar o seu cardápio e horários de forma prática.

## Estrutura
Ao iniciar este site, utilizei o arquivo reset.css para deixar todas as tags zeradas, porém decidi retirar para ter mais liberdade e não ter muito trabalho com o CSS.

Na página da Home utilizei o bootstrap para adicionar o Navbar(cabecalho), os ícones das redes sociais no footer e para mais alguns pequenos detalhes (O Header e Footer desta página foi utilizada em todas as outras).

Na página de Cardápio, adicionei um layout de card do bootstrap, porém fiz diversas modificações pelo compromisso com as regras de negócio. O card contém imagem, uma breve descrição e um hr para dividir a descrição do valor do produto.
